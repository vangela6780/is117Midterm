# Thread & Trace Design System

## Core Direction
- Style: Bauhaus with a museum editorial lens.
- Mission: Shift the narrative from waste guilt to creative reclamation.
- Archetype: The Creator.
- Persuasion principle: Cialdini's Social Proof.

## Visual Rules
- Use a grid-first layout with visible structure and hard edges.
- Pair earthy tones with industrial neutrals.
- Keep typography bold, direct, and legible.
- Contrast the Atacama problem space with vibrant, constructed gallery color.

## Tone Rules
- Vocabulary: reclaim, stitch, narrative, uniqueness, legacy.
- Avoid retail language, guilt framing, and vague sustainability clichés.
- Each section should read like an exhibit label or curator note.

## Component Rules
- Keep components small and single-purpose.
- Every content block should serve one narrative role: shock, cost, art, social proof, or action.
- Include explicit provenance or interpretation where possible.
